# Verification Reports

This directory contains JSON logs from protocol verification runs.

## Files

- `verification-log-testnet.json` - Testnet verification results
- `verification-log-fork.json` - Local fork verification results
- `verification-log-mainnet.json` - Mainnet verification results (if applicable)

## Usage

Generated by:
```bash
npx tsx scripts/verify-protocol.ts --network=testnet
npx tsx scripts/verify-protocol.ts --network=fork
npx tsx scripts/verify-protocol.ts --network=both
```

## Log Format

Each log file contains:
- Network info
- Contract addresses
- Wallet addresses
- Phase-by-phase results with checks and transactions
- Summary statistics
